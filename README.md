# internship-5-elevate-labs
Malware Types &amp; Behavior Analysis .
This project focuses on understanding malware concepts and performing basic malware analysis using online tools such as VirusTotal and ANY.RUN (free tier).
Objectives
Learn different types of malware
Analyze malware using hash-based analysis
Understand malware behavior and lifecycle
Identify malware spread methods
Suggest prevention techniques
Tools Used
VirusTotal – Malware detection and analysis
ANY.RUN (Free Tier) – Interactive malware sandbox
Key Topics Covered

What is malware

Difference between virus and worm

Ransomware basics

Malware spreading techniques

Malware prevention methods

Analysis Steps

Study malware types (virus, worm, trojan, ransomware)

Use known malware hashes

Analyze detection reports on VirusTotal

Observe behavioral indicators

Understand malware lifecycle

Identify spreading techniques

Recommend prevention measures

Summarize findings

Outcome

This project helped build foundational knowledge of cybersecurity threats and malware analysis techniques used in real-world environments.
